# Some Linux Setting Knowledge Base

This repository summarizes common Ubuntu/Linux server maintenance and GPU usage guides for quick lookup.

## Contents Overview

- **Disk mounting**: Identify, mount, and unmount newly inserted drives.  
  Document: `Disk_mounting/ubuntu_server_disk_mounting.md`

- **GPU settings and usage**:
  - PyTorch GPU checks and memory behavior.  
    Document: `GPU_setting/pytorch_gpu_operational_guidelines.md`
  - TensorFlow GPU checks and memory growth configuration.  
    Document: `GPU_setting/tensorflow_gpu_operational_guidelines.md`

- **Disk quota**: Enable, configure, and verify quotas for `/home`.  
  Document: `Quota_setting/ubuntu_server_quota.md`

- **NVIDIA driver reinstall**: Secure Boot notes and clean reinstall steps for driver 535.  
  Document: `Driver_reinstall/Driver_535.md`

- **TensorFlow environment setup**: Create a TF 2.15 + Python 3.11 environment and install packages.  
  Document: `Install_Tensorflow_on_linux/TF 2.15 + Py 3.11.md`

- **User/group and permissions**: Manage users, groups, permissions, and sudo access.  
  Document: `User_add/ubuntu_server_user_group_permissions.md`
